Problem: Gateway to Fest Fun

Rishith has successfully completed several challenges and earned a certain number of tokens.

These tokens are the key to enjoying the college fest, which offers multiple events where participants can win exciting cash prizes and gifts.

Each event requires a specific number of tokens to register freely.

Your task is to help Rishith determine how many events he can access based on the tokens he has earned and have fun by playing or participating in these events.

Input Specification:

  • input1: An integer T, representing the number of tokens Rishith has earned.
  • input2: An array of integers arr of length N, where each element represents the number of tokens required to access a specific event.

Output Specification:

Return an integer value representing the number of events Rishith can participate in.

Constraints:

  • 1 ≤ T ≤ 105
  • 1 ≤ N ≤ 1000
  • 1 ≤ arr[i] ≤ T

Example 1:

Input:

10
3 2 5 7 1 4

Output:

4

Example 2:

Input:

5
2 5 6 1

Output:

3

Example 3:

Input:

8
4 9 1 3

Output:

3
